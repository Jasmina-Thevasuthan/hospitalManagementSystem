# MediCare Hospital Management System

A desktop-based Hospital Management System developed using Python and Tkinter. The application helps manage patients, doctors, appointments, treatments, billing, and hospital reports through an intuitive graphical user interface.

## Project Overview

MediCare Hospital Management System is designed to streamline hospital administration by providing a centralized platform for managing patient records, doctor information, appointments, treatments, and billing operations.

The system includes PDF invoice generation and real-time analytics for better operational management.

---

## Features

### Patient Management
- Add new patients
- Update patient information
- Delete patient records
- Search patients by ID or name

### Doctor Management
- Register doctors
- Update doctor details
- Remove doctor records
- Search doctor information

### Appointment Management
- Schedule appointments
- Update appointment details
- Delete appointments
- Link patients with doctors

### Treatment Management
- Record treatments
- Update treatment information
- Manage treatment costs
- Track treatment history

### Billing & Invoice System
- Generate treatment invoices
- Export invoices as PDF
- Automatic cost calculation
- Professional invoice formatting

### Analytics Dashboard
- Total patient count
- Total doctor count
- Treatment statistics
- Revenue calculation
- System status overview

### Search Functionality
- Real-time search
- Filter records by ID or name
